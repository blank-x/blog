##### 代码换行问题解决

设置Prettier: Print Width 指定每行代码的最佳长度， 如果超出长度则换行。 

设置成800



#### 缩进设置

在设置中搜索tabSize:2;

如果使用了prettier，需要设置Prettier: Tab Width为2



### 项目中重写console.log

开发环境中不打印；

生成环境中log

```
console.log = (function (oriLogFunc) {
  return function () {
    //判断配置文件是否开启日志调试
    if (!Config.isProduct) {
      try{
        oriLogFunc.call(console, ...arguments);
      }catch(e){
        console.error('console.log error', e);
      }
    }
  }
})(console.log);
```
